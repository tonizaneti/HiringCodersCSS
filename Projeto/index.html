TECNOLOGIAS & SUAS RESPECTIVAS LINGUAGENS

React Native é uma biblioteca Javascript que foi criada e é atualmente mantida pelo Facebook. Utilizada para o desenvolvimento de aplicativos para dispositivos móveis, Android e IOS, de forma nativa.

Seus princípios de funcionamento são idênticos ao do ReactJS, com a diferença apenas que não manipula o DOM através do Virtual DOM, mas sim executando num processo em segundo plano (interpretando o Javascript escrito pelos desenvolvedores) diretamente no dispositivo final, e se comunicando com a plataforma nativa por meio de uma serialização, uma ponte assíncrona e em lote.

O React Native não usa HTML, em seu lugar, trabalha com Javascript puro e tendo com sintaxe JSX.

O MySQL é um sistema de gerenciamento de banco de dados (SGBD), o qual utiliza a linguagem SQL (Linguagem de Consulta Estruturada, Structured Query Language), como interface. Sendo atualmente um dos SGBD´s (Sistemas Gerenciadores de Banco de Dados) mais populares da Oracle Corporation, com mais de 10 milhões de instalações pelo mundo.

Podemos destacar entre seus usuários: Nasa, Friendster, Banco Bradesco, Dataprev, HP, Nokia, Sony, Lufthansa, U.S. Army, U.S. Federal Reserve Bank, Associated Press, Alcatel, Slashdot, Cisco Systems, Google, entre outros.

PHP é a sigla para "PHP Hypertext Preprocessor", ou como em seu início Personal Home Page, é uma linguagem interpretada livre, sendo desenvolvida inicialmente apenas para o desenvolvimento de aplicações presentes do lado do servidor, capazes de gerar conteúdo dinâmico na WWW. Está entre as primeiras linguagens passíveis de inserção em documentos HTML, dispensando em muitos casos o uso de arquivos externos para eventuais processamentos de dados. O código é interpretado no lado do servidor pelo módulo PHP, que também gera a página web a ser visualizada no lado do cliente. A linguagem evoluiu, passou a oferecer funcionalidades em linha de comando, e além disso, ganhou características adicionais, que possibilitaram usos adicionais do PHP, não relacionados a web sites. É possível instalar o PHP na maioria dos sistemas operacionais, gratuitamente. Concorrente direto da tecnologia ASP pertencente à Microsoft, o PHP é utilizado em aplicações como o MediaWiki, Facebook, Drupal, Joomla!, WordPress, Magento e o Oscommerce.
